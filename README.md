# Matlab Animations / Animated plots

Animating different experiments in Matlab.

## Experiments

- Spring mass system
- Hub Magnet
- Tank system
- Two-Mass-Spring System
- Spring-Mass System

## Simulate

For each experiment a 'test*' script is provided to run / test
the animated plot with arbitrary data.
